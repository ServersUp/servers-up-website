ServersUp

A Discord bot that posts when a game server goes UP or DOWN. Alerts only on status changes—not on a timer in your channel.

Add to Discord

How it works

  1. Add the bot to your server.
  2. Run /subscribe with game, region, and server (autocomplete at each step).
  3. When status changes, the bot posts in that channel.

Commands

  • /subscribe — add a subscription (Manage Channels or Administrator required)
  • /unsubscribe — remove one
  • /subscriptions — list by channel
  • /status — check UP/DOWN now
  • /regions and /servers — browse the catalog
  • /games and /help

Supported games

Select a game and region, then search. WoW uses us/eu/kr/tw; FFXIV uses na/eu/jp/oce.
